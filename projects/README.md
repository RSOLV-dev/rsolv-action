# Active Projects

This directory contains working documents for in-flight projects. Each project gets its own subdirectory with a README and related tracking documents.

## Active Projects

- `billing-integration-2025-10/` - Integration of RFCs 065-068 (Customer Onboarding, Billing, Action Polish, Testing Standards)

## Archive Policy

When a project completes:
1. Move the project directory to `archived_docs/`
2. Transfer permanent knowledge to ADRs and permanent documentation
3. Delete temporary/obsolete content

Example:
```bash
mv projects/my-project archived_docs/my-project
```
