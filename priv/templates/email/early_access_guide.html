How RSOLV Works

RSOLV combines security analysis with AI to automatically fix issues in your codebase. Here's the process:

1. Security Analysis

RSOLV scans for over 80 security patterns across JavaScript, TypeScript, Python, Ruby, and Java to identify vulnerabilities.

2. AI-Powered Solution

Our AI generates secure fixes with comprehensive documentation explaining the vulnerability and solution.

3. Pull Request Creation

A well-documented PR is created with the fix, security impact assessment, and educational content.

4. You Deploy, You Pay

Review and merge the PR when ready. You're only charged $15 when the fix is deployed (after 10 free fixes).

How to Write Effective Issues

The quality of RSOLV's solutions depends on the clarity of your issues. Here are some best practices:

A Great Issue Example:

Title: Fix pagination error in user list
Label: rsolv:automate

Description:
When navigating to page 2 of the user list, the app shows the same users from page 1.

Steps to reproduce:
1. Go to /users
2. Click the "Next Page" button
3. Observe that the same 10 users are displayed

Expected behavior:
Page 2 should show the next 10 users

Related files:
- src/components/UserList.js
- src/api/users.js

Be Specific

Clearly describe what's broken and how it should work instead.

Include Context

Mention which components, files, or functions are involved.

Show Steps

List step-by-step instructions to reproduce the issue.

Use the Right Label

Always use the "rsolv:automate" label to trigger RSOLV.

Setup Instructions

Getting started with RSOLV is simple. Add this GitHub Action workflow to your repository:

name: RSOLV Automation

on:
  issues:
    types: [labeled]

jobs:
  resolve:
    if: contains(github.event.issue.labels.*.name, 'rsolv:automate')
    runs-on: ubuntu-latest
    steps:
      - uses: rsolv-dev/rsolv-action@v1
        with:
          github-token: ${"{{ secrets.GITHUB_TOKEN }}"}
          rsolv-api-key: ${"{{ secrets.RSOLV_API_KEY }}"}

For detailed setup instructions, visit our getting started guide.
