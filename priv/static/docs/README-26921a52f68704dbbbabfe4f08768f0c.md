# Static Documentation

This directory contains publicly accessible documentation for the RSOLV landing site.

## Source of Truth

The authoritative source for these documents is in the RSOLV-docs repository:

- User guides: `/RSOLV-docs/user-guides/`
- Templates: `/RSOLV-docs/templates/`
- Business documentation: `/RSOLV-docs/business/`

## Updating Documentation

When updating documentation:

1. Make changes in the RSOLV-docs repository first
2. Copy the finalized documents to this static docs directory
3. DO NOT edit documents directly in this directory

This prevents duplication and ensures consistent documentation.
